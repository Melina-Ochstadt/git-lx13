WORKING DIRECTORY

Area de Trabajo, donde rara vez vamos a estar utilizando la consola y hace referencia a la carpeta donde se inicializo el proyecto de git. Aqui basicamente vamos a crear, modificar y eliminar los diferentes archivos que formen parte de nuestro proyecto.

  • dir || ls muestra el contenido de la carpeta actual
  • mkdir <name> crea una carpeta con el nombre especificado
  • cd <path> abre el directorio seleccionado
  • rm -rf .git eliminamos el repositorio de git, -rf lo hace de manera forzada
